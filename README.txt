== Description ==

This plugin for Wordpress creates scroller post type which could be edited with any post type editor. Scroller
post type has background metabox with color selector. The plugin attaches selected color as  background style 
in the public side to body via javascript. The background color transition is animated using jQuery Color. WPBakery
component styles are supported on the public page. 

A shortcode nm_scroller is available for inserting scroller to a page so only single scroller collecting all
posts from cpt is available.
